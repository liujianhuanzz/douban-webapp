# webapp

利用豆瓣开放API做的电影和音乐的相关内容。
+ 支持显示最近上映的电影，点击列表显示电影的详细信息
+ 支持音乐人、作品搜索，点击列表，显示相关专辑的详细信息
+ vue/vuex/vuex-router

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

```
